# PureMath

The purest math package!

## How to install:

To install this package, run ```pip install puremath```

## Tutorial

Some things that this package includes is the add & subtract function. Here's an example of the add:

```
import puremath as pm

pm.add(4, 1) #result is 5
```

And to subtract, it's as easy as

```
import puremath as pm

pm.subtract(4, 1) #result is 3
```
